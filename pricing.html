Pricing FAQ

What happens if I cancel?

Your agents keep running for a 7-day grace period. After that, they go dormant. We never delete your data—re-subscribe any time to wake them up. Just note that there may be up to a 24-hour delay between re-subscribing and agents restarting.

What are the hidden costs?

You’ll need your own LLM (Anthropic, OpenAI, etc.) API key to power the team. You'll only pay for what the agents actually use.

What API keys do I need?

We recommend Anthropic for the brains. Specifically, the latest Sonnet model is well suited for the Tech Lead and QA, and the highest power Opus has worked well for the Developer. We have not tested every possible combination, nor have we tested other models for these use cases. Please let us know what is working well for you! Also, you're not locked into a model. We like Anthropic, but do you.

What are the system requirements?

Each team requires a small VPS with Docker-friendly Linux, outbound HTTPS, and approximately 8GB of RAM. See the setup manual for detailed specs.

How much does it cost to run?

It depends. You can set monthly limits in-product and with the LLM provider to control costs. However, ticket volume, project complexity, and a plethora of other factors influence cost.
